# 默认题库

使用项目的一键启动脚本后，网页会自动连接当前 `默认题库` 目录，不需要手动选择。顶部“已连接”用于重新扫描，“切换题库”用于打开其他本地题库目录。

默认目录当前包含实际导入的题库。将题目图片和答案图片放入题库对应的普通文件夹，然后在网页中点击“已连接”重新扫描。

```text
Q-01-1-01.png       第 1 章、第 1 小节、第 1 题的题目图
Q-01-1-01.1.png     同一道题由多张图片组成时的第 1 张
Q-01-1-01.2.png     同一道题由多张图片组成时的第 2 张
A-01-1-01.png       第 1 章、第 1 小节、第 1 题的答案图
```

网页连接后会自动生成 `题库数据.json`。该文件只保存题库结构、重命名和目录映射；学习标记保存在项目根目录的 `用户数据/用户数据.json`，不会混入题库项目数据。
