# 更新日志

本项目遵循语义化版本号。完整下载包和每个版本的说明见 [GitHub Releases](https://github.com/EnderRayven/NPEElearningtool/releases)。

## v0.3.6 · 复习模块与题库更新

- 移除复习弹窗底部重复的掌握情况按钮，将上一题、题号和下一题导航居中显示。
- 新增“机械设计·讲义课后习题”题库，默认题库更新为 22 个题库、7139 道题。
- 软件包与题库数据包继续在同一个 Release 中分别提供，便于独立更新。

## v0.3.5 · 复习记录与手写工具优化

- 支持在复习弹窗中删除单次复习记录，并自动修复后续复习的状态链。
- 为重置复习记录、清空手写笔记和删除复习记录增加二次确认，降低误操作风险。
- 优化题目切换后的滚动定位、学习看板的“新题/复习”日历展示和复习状态写入。
- 手写笔记和草稿画布改进套索选择、删除、移动、平滑绘制、画布收缩与指针交互。
- 修正部分数学/英语答案与机械题图资源；默认题库更新为 21 个题库、7068 道题。
- 软件包与题库数据拆分为两个 Release 资产，便于单独更新应用或题库。

## v0.3.4 · 工具箱与题目笔记体验更新

- 新增工具箱，提供专注计时器、计时器小窗、全屏显示和集中式题目笔记入口。
- 学习看板支持题目预览、笔记查看、复习记录重置和新题/复习结果分层统计。
- 新增教材习题题库与课后习题资源，统一机械专业课题库目录命名并补充题图。
- 完善计时器、题目笔记、复习记录和学习活动测试覆盖。
- 将教材原始题图留在本地生成流程中，发布仓库只保留默认题库资源。

## v0.3.3 · 手写画布与滚动体验优化

- 修复缩放、移动和扩展画布时的笔迹坐标问题，选中笔迹变换更稳定。
- 新增插入空间功能，支持向下拖动为笔迹内容预留空白。
- 书写接近底部时自动延长画布，擦除或删除底部笔迹后自动收缩，最低恢复默认高度。
- 优化粗细调节、实时笔迹大小预览、四字工具按钮排布和手写工具栏样式。
- 页面和导航区域预留滚动条空间，统一滚动条样式，避免内容因滚动条出现而横向跳动。

## v0.3.2 · 数学真题与笔记体验更新

- 新增并完善数学二历年真题题库，题目与解析图片随题库目录保存，补充 2018 年 9 月真题内容。
- 支持整卷与考点目录切换，考点目录按原网站层级展示，并提供按年份和题号的快速导航。
- 优化目录层级、字号、题号导航和页面内滚动布局，避免长目录或导航模块超出可视范围。
- 手写笔记工具栏调整为“橡皮、画笔、套索”，快捷键为 `1`、`2`、`3`，并支持进入手写模式后直接使用。

## v0.3.0 · 正式版

- 新增 2004–2026 年英语一真题，并按年份和题型优化侧栏导航。
- 完善英语真题答案与解析，修复机械专业课题图、答案图和多图题显示。
- 新增题目笔记、可编辑手写、多轮复习、学习日历与分层进度看板。
- 完善图片题库导入、默认工作区同步、完整备份和题目导出体验。
- 在应用顶部加入 GitHub 入口，更新界面截图、项目介绍与下载指引。
- 正式发布统一命名的完整包 `NPEE-Study-Space-v0.3.0-Full.zip`，保留 macOS 与 Windows 一键启动入口。

## v0.2.0
